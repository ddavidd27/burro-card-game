A fun, multiplayer card game built with WebSockets where 2–10 players compete to avoid spelling "BURRO"!

## About

Burro is a lively online game where players pass Spanish cards (e.g., 7 de espadas) and shout "Burro!" to score points. Features include:

- **Multiplayer**: 2–10 players via WebSockets.
- **Gameplay**: Pass cards, collect sets, shout "Burro" when someone has four-of-a-kind.
- **Scoring**: Earn letters (B-U-R-R-O); spell "BURRO" and you're out!
- **UI**: Brownish theme, responsive for mobile, with card images and rules modal.
- **Real-Time**: Names (e.g., Cristina, David) and actions update instantly.

## How to Play

- **Join**: Enter your name and create/join a room.
- **Pass**: Swap cards with others, but don’t hold five!
- **Shout**: Click "Shout Burro!" when someone has four matching cards.
- **Win**: Be the last player standing without "BURRO".
- Check the in-game rules for details!
